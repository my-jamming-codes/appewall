## Introduction

Validation is a crucial part of application development to ensure data integrity and correctness. The Micronaut framework provides robust support for validations including `@NotNull` and various numeric checks.

## Setting Up Validation

To start using validation in Micronaut, you need to add the necessary dependencies to your project. Here’s an example using Maven:

```xml
<dependency>
  <groupId>io.micronaut</groupId>
  <artifactId>micronaut-validation</artifactId>
  <version>2.5.13</version>
</dependency>
```

## Using `@NotNull` and Numeric Checks

Micronaut supports a wide range of validation annotations. Here are examples of how to use `@NotNull` and numeric checks:

### `@NotNull` Example

```java
import javax.validation.constraints.NotNull;

public class User {
    @NotNull
    private String name;

    // getters and setters
}
```

### Numeric Checks Example

```java
import javax.validation.constraints.Min;
import javax.validation.constraints.Max;
import javax.validation.constraints.Positive;

public class Product {
    @Min(1)
    private int quantity;

    @Max(100)
    private int discount;

    @Positive
    private double price;

    // getters and setters
}
```

## Applying Validations in Controllers

You can apply validations directly in your Micronaut controllers. Here’s an example:

```java
import io.micronaut.http.annotation.*;
import javax.validation.Valid;

@Controller("/users")
public class UserController {

    @Post
    public User createUser(@Body @Valid User user) {
        // Handle user creation
        return user;
    }
}
```

In this example, the `@Valid` annotation ensures that the `User` object passed to the `createUser` method is validated according to the annotations defined in the `User` class.

## Conclusion

Micronaut makes it easy to implement validations in your application. By using the provided annotations, you can ensure that your application handles data correctly and prevents invalid data from causing issues. Whether you’re using `@NotNull`, numeric checks, or other validation annotations, Micronaut’s support for validation is comprehensive and straightforward to use.
